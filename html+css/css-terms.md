- basic selector
- attribute selector
- pseudo-selector
- pseudo-element 
- `type selector`

	`div`와 같은 요소 명을 이용하여 요소를 선택
	
- `class selector`

	`.`을 이용하여 특정 클래스명을 가진 요소를 선택
	
- `ID selector`

	`#`을 이용하여 특정 ID명을 가진 요소를 선택
	
- `universal selector`

	`*`을 이용하여 모든 요소를 선택 
	
-  simple selector

	어떠한. basic selecor, attribute selector, pesudo-selector, pesudo-element selector를 하나만가지고 있는 경우
	
-  combinator

	DOM node들간의 관계를 이용하여 요소를 선택
